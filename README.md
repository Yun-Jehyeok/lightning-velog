# Velog Clone Project

Aura Component(FE) + Apex(BE) + SLDS 를 이용한 Velog 클론 프로젝트입니다.

## 기능
- 회원가입
- 로그인
- 로그아웃
- 게시글 작성, 삭제, 출력
- Dropdown, Dynamic Navigation
- 등등 게시글의 전반적인 동작

## 개선해야 할 사항
- 토큰 생성
- 비밀번호 암호화
- 댓글
- 게시글 수정
- 회원정보 수정
- 회원 탈퇴
- 사진 및 이미지 처리
- 등등 개선해야 할 사항이 많습니다.

## 앞으로..

애초에 Aura Component에 익숙해지기 위해 사용했던 프로젝트이기도 하고 Aura 보다는 LWC를 이용해야하기 때문에 추가적으로 무언가 하지는 않습니다.
